Generate following particles

Random decaying particles which follow the mouse cursor.

        
          const shapeInstance = new Shape('canvas');
          shapeInstance.generateFollowingCircles({
            n: 1000,
            maxRadius: 50,
            colours: shapeInstance.colours.fogUnderWhiteSky,
          })

Generate bouncing balls

Random bouncing circles which simulate the effect of gravity.

Click on the canvas to restart the animation.

        
          const shapeInstance = new Shape('canvas');
          shapeInstance.generateBouncingCircles({
            n: 100,
            minRadius: 10,
            maxRadius: 20,
            colours: shapeInstance.colours.colors02
          })

Generate Julia Fractals

Find an interesting frame by moving your cursor slowly.

Click once to freeze the frame. Click again to zoom in.

        
          const shapeInstance = new Shape('canvas');
          shapeInstance.generateJuliaFractals()

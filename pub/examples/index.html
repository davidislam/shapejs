Home Examples API

Generate following particles

Random decaying particles which follow the mouse cursor.

        
          const shapeInstance = new Shape('canvas');
          shapeInstance.generateFollowingCircles({
            n: 1000,
            maxRadius: 50,
            colours: shapeInstance.colours.fogUnderWhiteSky,
          })

Generate bouncing balls

Random bouncing circles which simulate the effect of gravity.

Click on the canvas to restart the animation.

        
          const shapeInstance = new Shape('canvas');
          shapeInstance.generateBouncingCircles({
            n: 100,
            minRadius: 10,
            maxRadius: 20,
            colours: shapeInstance.colours.colors02
          })

Generate Julia Fractals

Find an interesting frame by moving your mouse cursor slowly.

Click once to freeze the frame. Click again to zoom in.

        
          const shapeInstance = new Shape('canvas');
          shapeInstance.generateJuliaFractals()

Generate Mandelbrotset Fractals

Click to zoom in. Shift-click to pan. Alt-click to zoom out.

        
          const shapeInstance = new Shape('canvas');
          shapeInstance.generateMandelbrotSetFractals({
            maxIterations: 500
          })

Generate Amplifying Rectangles

Rectangles respond to your mouse cursor.

        
          const shapeInstance = new Shape('canvas');
          shapeInstance.generateAmplifyingRectangles({
            n: 25,
            colours: shapeInstance.colours.rariSeatOrange,
            minHeight: 150,
            maxHeight: 350
          })

Generate Colliding Particles

Hovering over them reveals their true colour. Try it!

        
          const shapeInstance = new Shape('canvas');
          shapeInstance.generateCollidingCircles({
            n: 100,
            radius: 10,
            colours: shapeInstance.colours.colourful,
            range: 120,
            speed: 3
          })

Generate Moving Shrinking Circles

Hovering over circles causes them to shrink. Try it!

        
          const shapeInstance = new Shape('canvas');
          shapeInstance.generateInteractiveCircles({
            n: 250,
            colours: shapeInstance.colours.colorThemesky,
            animated: true,
            speed: 0.2,
            range: 100
          })

Generate Static Circles

Could be your next wallpaper!

Click to regenerate.

        
          const shapeInstance = new Shape('canvas');
          shapeInstance.generateCircles({
            n: 200,
            colours: shapeInstance.colours.fogUnderWhiteSky
          })

Generate Rectangles

More wallpaper options :)

Click to regenerate.

        
          const shapeInstance = new Shape('canvas');
          shapeInstance.generateRectangles({
            n: 200,
            colours: shapeInstance.colours.rariSeatOrange,
            maxWidth: 150,
            minWidth: 50,
            maxHeight: 425,
            minHeight: 100
          })
